Construct a new MicroMouse object.

int  l_y

This function computes the path from the mouse's current position to the goal recursively using the Depth First Search algorithm. It is called in the search_maze() function and accounts for backtracking.

Parameters
l_xinteger Local x index from which the DFS has to be computed to reach the goal
l_yinteger Local y index from which the DFS has to be computed to reach the goal
Returns
true If there is a path found from the (l_x, l_y) node to the goal node
false If there is no path found from the (l_x, l_y) node to the goal node
